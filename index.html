Reflection: Building Websites with AI

I used AI to brainstorm section structure, generate a baseline layout, and debug CSS issues faster. The key value wasn’t “writing code for me” — it was accelerating iteration: propose → test → fix. I still had to validate responsiveness, correct broken paths/links, and keep the design consistent.

Reflection: Interview Question

If asked “How do you build a simple site end-to-end?”, I’d explain: define the sections, write semantic HTML, style with reusable CSS patterns, test on mobile/desktop, then deploy on GitHub Pages and verify the live URL. I’d highlight that deployment and testing are part of the job, not an afterthought.

Reflection: LinkedIn Post

This week I built and deployed a single-page portfolio using HTML/CSS and an AI-assisted workflow. Biggest takeaway: small details (spacing, link targets, responsive layout) create “polish.” Next: keep adding projects and leveling up responsive design.
